## 🌟 Features

- **🎮 12 Interactive Modules** - Progressive learning chapters covering blood science
- **🌍 Bilingual** - Full English and French language support
- **🎯 Gamified Learning** - Memory games, quizzes, drag-and-drop activities, and simulations
- **📱 Mobile-Friendly** - Responsive design works on phones, tablets, and desktops
- **🏆 Achievement System** - Earn stars, badges, and a completion certificate
- **🔬 Educational Content** - Accurate blood type compatibility and genetics information
- **⚡ Zero Dependencies** - Single HTML file, no build step required
- **🖼️ PWA Ready** - Can be installed as a progressive web app

## 🚀 Quick Start

No installation or build process needed! Simply open `index.html` in any modern web browser:

```bash
# Clone the repository
git clone https://github.com/your-username/blood-adventure-academy.git
cd blood-adventure-academy

# Option 1: Open directly in browser
open index.html

# Option 2: Serve with any static server
python -m http.server 8000
# Then visit http://localhost:8000
```

## 🌐 Deployment

This is a static site that can be deployed anywhere:

- **GitHub Pages** - Free hosting directly from this repo
- **Netlify** - Drag & drop the folder
- **Vercel** - Connect your GitHub repo
- **Any CDN** - Upload the files
- **Local File** - Works from `file://` protocol (limited functionality)

### Deploy to GitHub Pages

1. Go to **Settings > Pages** in your GitHub repo
2. Select **Deploy from a branch**
3. Choose the `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live at `https://your-username.github.io/blood-adventure-academy/`

## 🩺 Educational Accuracy

The blood type information in this application is based on established medical knowledge:

- **ABO Blood Group System** - The four main blood types (A, B, AB, O)
- **Rh Factor System** - Positive (+) and negative (-) classification
- **Transfusion Compatibility** - Who can donate to whom
- **Genetic Inheritance** - Simplified Mendelian model for educational purposes

> ⚠️ **Note:** The inheritance calculator uses simplified genetics and does not account for rare subtypes or the Bombay phenotype.
